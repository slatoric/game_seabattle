# game_seabattle
The "Sea Battle" game (php)

Genre: step-by-step strategy for one player (vs computer)

How to install:
1. Unpack archive
2. Put "sea.php" and "pic" directory on site root or subdirectory
3. Open browser and go <site name>/sea.php
